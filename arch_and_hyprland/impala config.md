# Impala Configuration

## Overview
Impala is a modern, feature-rich music player daemon client. It provides a beautiful interface for controlling music playback.

## Configuration Files
- `~/.config/impala/config.toml`: Main configuration file in TOML format.

## Setup Commands
No special setup commands are required for impala beyond ensuring the package is installed and having a music player daemon running (like mpd).

## Why This Configuration
- Impala is chosen for its modern interface and rich feature set for music control.
- The configuration is kept minimal, focusing on essential settings for usability.
- It integrates well with music player daemons like mpd for a complete music listening experience.

## Installed Packages
Impala package is required. See the native packages list for the exact version.

## Usage
Launch impala to control your music player daemon. It provides controls for playback, playlists, and library browsing.
